1.1 Tic-Tac-Toe Java Game

A feature-rich Tic-Tac-Toe game built with Java Swing featuring persistent score tracking, player customization, and historical records.

1.2 Features
- Classic 3x3 Tic-Tac-Toe gameplay
- Customizable player names
- Real-time score tracking
- Persistent score history
- Best scores hall of fame
- Clean Java Swing interface

1.3 Technologies Used
- Java 8+
- Java Swing (GUI)
- File I/O for data persistence

1.4 Installation & Usage

Prerequisites
- Java JDK 8 or higher

Running the Game
1. Download the "TicTacToeSimple.java" file
2. Compile: "javac TicTacToeSimple.java"
3. Run: "java TicTacToeSimple"

How to Play
>Enter player names in the text fields
>Click any empty cell to place your symbol (X or O)
>First player to get 3 in a row (horizontal, vertical, or diagonal) wins!
>View best scores by clicking "View Best Scores" button
